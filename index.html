PHP Dynamic Web App deployment on AWS

This project demonstrates the end-to-end deployment of a dynamic PHP-based web application on AWS infrastructure. I started by launching an EC2 instance with Ubuntu and configured networking through security groups and Elastic IP. Apache and PHP were installed on the server, and MySQL was set up to manage the backend database. Static content was managed using S3, and FileZilla was used to securely upload the project files. SSH access was configured via PuTTY using a private key. I configured the Apache server to point to the uploaded project directory and integrated the database with proper user privileges. To optimize performance and access, additional AWS services like Route 53 (domain mapping), CloudFront (CDN), and IAM roles (security) were considered or partially implemented. The final result was a fully functional, accessible dynamic website hosted on the cloud.

Tech: AWS EC2, Apache2, PHP & MySQL, PuTTY, FileZilla, Ubuntu (Linux)
